#  SocialHub – Modern Social Media UI

A fully interactive social media interface built using **HTML** and **JavaScript**.

The project simulates a real-world social platform with posts, likes, comments, profile editing, notifications, and more — all without any backend.

---

##  Features

###  **Feed Page**
- View dynamic posts.  
- Like/unlike posts.  
- Comment on posts.  
- Save/unsave posts.  
- Smooth scroll to any selected post.

###  **Profile Page**
- Edit profile info (name – username – bio).  
- View created posts.  
- View saved posts.   
- Switch between tabs (Posts – Saved – Followers).

###  **Notifications**
- Dynamic notification list.  
- Clickable notifications that lead to the related post.

###  **Search Page**
- Live search for users.  
- Follow/Unfollow users directly from search.

###  **Create Post Modal**
- Write a new post.  
- Auto-insert the new post into feed + profile.
